# ezbs64
https://github.com/siriusjbk/ezbs64
bs64 사용을 사이트에서 인코딩 디코딩 하는것 보다 빠르게 하기 위해 제작했음
귀찮아서 개빠르게 코딩한거라 문제 많음

엔터만 누를시 클립보드 내용이 자동으로 디코딩 된후 결과값이 복사됩니다.
ex)
#clipboard : dGVzdA==
>>> #pause enter
test
가 복사되었습니다.
아무키나 눌르면 종료
#clipboard : test

인코딩할 데이터를 입력후 엔터를 누를경우 데이터를 인코딩 한후 
ex)
>>>test
dGVzdA==
가 복사되었습니다
아무키나 눌르면 종료
#clipboard : dGVzdA==

e 만을 입력시에는 클립보드에 있는 데이터를 자동으로 인코딩 한후 결과값이 복사됩니다.
ex)
#clipboard : test
>>>e
dGVzdA==
가 복사되었습니다
아무키나 눌르면 종료
#clipboard : dGVzdA==
